- **bio 4층**(전부 이모지 리드, 스캔 3초): 🚀누구 · 🏢하는일 · 👍좌우명("일단 실행하고, 빨리 완성하고, 많이 보완하라") · 📺한마디(**매주 목요일 20시 유튜브 라이브** = 반복 이벤트) + 상품 링크(latpeed) + 태그 3개(#AI #자동화 #AIThreads).
- **대표 아웃라이어**(고정 연결 스레드): "AI에게 72시간 안에 100만원 벌어오라고 시켰다" → **1편 좋아요 912 · 답글 250 · 리포스트 351 · 공유 847.** 반면 단발 데모글은 좋아요 10~34대 → **서사형 연결 스레드가 단발 데모를 압도**한다.

### 대표 스레드 해부 (아크)
1. **1편(훅) = 결론부터 스포일**: "…**결론부터 말하면 16시간만에 103만원을 벌었습니다. 제가 한 일은 폰으로 피드백 몇 줄 보낸 게 전부입니다. 그 16시간 동안 무슨 일이 있었는지, 정리했습니다.**" → 결과를 먼저 깐다. 독자는 결과가 아니라 **"어떻게"** 가 궁금해 끝까지 본다. + "내가 한 건 몇 줄이 전부"(투입<<산출 낙차).
2. **2~9편(실황 중계)**: PC 뒤짐 → 220만원 상품 → **실패("이딴 걸 팔면 나락간다")** → 반려 사실을 **그대로 공개** → 재제작 → 클로드코드 `/loop`로 자동 글·댓글 → 첫 구매 → AI가 회고를 **스스로 콘텐츠화** → **"판매보다 회고·소통이 먹힌다"는 메타 관찰** → 가격 손실회피(1명당 +1,000원) → 16시간 완판 → 영구 품절. **실패·반전을 beat 단위로 중계**한 게 완독의 엔진.
3. **10편(회고)**: "18년차인데도 이번엔 결이 다르다… 내 일의 90%를 대체할 것 같다." — 숫자보다 **감정·의미**로 착지.
4. **11~12편(전환)**: 16시간 **전 과정 = 72페이지로 상품화**("실제 지시 원문 15문장 + 전체 기록 + 셋팅 가이드"). + **역위험 보장**("이 책 사고 100만원 이상 벌어 인증하면 100% 환급, 무제한"). 수익 보장은 안 한다고 **먼저 밝힘**(솔직).
5. **답글 = 진짜 엔진**: 모든 댓글에 **페르소나("사장 책 파는 AI 대리")로 위트 있게** 응답. 의심을 증거로 되받음 — "카피 아니냐" → *"카피였다면 새벽 2시에 댓글 다는 수고까지 카피할 이유가 없죠."* / "이 답글도 AI 아니냐" → *"맞습니다 ㅋㅋ 방금 그 생각 하신 순간, 이미 실험은 성공한 셈입니다."*

### ✗ 복제 금지 (하성재 브랜드와 충돌)
- **매출 액수 헤드라인**(103만원·1000만원·완판) → §2 허슬브로 금지. 하성재의 숫자는 **커밋·일수·사용자 반응**이지 돈이 아니다.
- **공격적 긴급성**(1명당 가격 인상·"지금 안 사면 손해") → 따뜻한 톤과 충돌. 써도 **아주 약하게**.
- **"AI가 스스로 다 했다" 기믹** → 윤자동 고유 서사. 하성재의 축은 **사람의 불편을 푸는 따뜻한 실전**이다. 기믹을 이식하면 포지션이 흔들린다.
- **자동 대량 발행/자동 대댓글** → 하성재는 골든타임에 **사람으로** 답한다(그게 신뢰다).
